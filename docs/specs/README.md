# Blueprint Specification

This directory contains the Blueprint specification.

## Structure

The specification will be organized as follows:

```
docs/specs/
  README.md            This file
  overview.md          The specification at a glance
  vocabulary/          Definitions of each term in the vocabulary
  semantics/           Runtime behavior of each term
  schema/              JSON Schema for Blueprint documents
  conformance/         What it means for an engine to be conforming
  versioning.md        How Blueprint documents declare and evolve versions
```

The specification is currently in draft. Content will be added as the design stabilizes.
