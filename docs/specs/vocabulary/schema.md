# Schema

**Status:** Draft — placeholder. Content not yet written.

Part of the [Blueprint specification](../overview.md), version 0.1.0-draft.

## Purpose

Defines how Blueprint uses typed shapes to describe the payloads of channels,
the contents of states, the parameters of scopes, and the configuration of
compounds. Fixes JSON Schema 2020-12 as the schema language and establishes
the distinction between *logical shape* (described by the schema) and
*physical representation* (chosen by the engine).
