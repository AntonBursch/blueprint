# Blueprint (the artifact)

**Status:** Draft — placeholder. Content not yet written.

Part of the [Blueprint specification](../overview.md), version 0.1.0-draft.

## Purpose

Defines *Blueprint* as the top-level artifact: a portable, self-contained
document describing an application's compounds, scopes, channels, states,
marks, and signatures. This document fixes the required and optional fields
of a Blueprint artifact and the rules by which it is recognized as
well-formed and valid.
