# Scope

**Status:** Draft — placeholder. Content not yet written.

Part of the [Blueprint specification](../overview.md), version 0.1.0-draft.

## Purpose

Defines scope as Blueprint's primary structural primitive. A scope is a named,
bounded region of execution within which compounds are instantiated, channels
are connected, states live, and capabilities propagate. Scopes form the
lifecycle and naming backbone of every Blueprint runtime.
