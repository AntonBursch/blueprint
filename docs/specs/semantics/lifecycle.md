# Lifecycle

**Status:** Draft — placeholder. Content not yet written.

Part of the [Blueprint specification](../overview.md), version 0.1.0-draft.

## Purpose

Defines the runtime lifecycle of a Blueprint: loading, entering the root scope,
instantiating compounds, connecting channels, initializing states, running,
and shutting down. Fixes the ordering guarantees every conforming engine MUST
provide.
