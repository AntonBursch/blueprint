# Blueprint examples

Worked examples of Blueprint artifacts. Each example is a valid JSON document
that conforms to the Blueprint schema and illustrates a specific set of
features.

Examples are informative. They are not part of the normative specification,
but every example published here MUST validate against the current schema.

| File | Illustrates | Status |
|---|---|---|
| hello-world.blueprint.json | Smallest useful Blueprint | Not yet authored |
| counter.blueprint.json | State, channels, a single compound | Not yet authored |
| dashboard.blueprint.json | Composition, nested compounds, imports | Not yet authored |

Examples will be authored in [Phase 5](../ROADMAP.md#phase-5--rationale-primers-examples).
